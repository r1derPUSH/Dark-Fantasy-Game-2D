# 🌐 Dark Fantasy Game

Це демо-проєкт, який містить HTML, CSS, JS і зображення. Гра створена на чистому JavaScript без використання бібліотек та фреймворків.

## 🧩 Структура

- `index.html` — головна HTML-сторінка
- `style.css` — стилі
- `main.js` — логіка (JavaScript)
- `img/` — зображення
